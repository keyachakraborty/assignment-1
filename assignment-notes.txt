HTML TAGS AND ATTRIBUTES USED

1. <!DOCTYPE html>

Declares that the document is written in HTML5.

2. <html lang="en">

Root element of the webpage.

Attribute:

 lang="en" specifies that the page language is English.

3. <head>

Contains metadata and page information.

4. <meta charset="UTF-8">

Defines the character encoding used in the webpage.

5. <meta name="viewport" content="width=device-width, initial-scale=1.0">

Makes the webpage responsive on different screen sizes.

6. <title>

Sets the browser tab title as "Student Course Enrollment Form".

7. <link rel="stylesheet" href="style.css">

Connects the external CSS file with the HTML page.

Attributes:

 rel="stylesheet" indicates a stylesheet file.
 href="style.css" specifies the CSS file path.

8. <body>

Contains all visible content displayed on the webpage.

9. <div class="container">

Groups all form elements together.

Attribute:

 class="container" is used for CSS styling.

10. <h2>

Displays the heading:
Student Course Enrollment Form

11. <form action="/action.php" method="post">

Creates the enrollment form.

Attributes:

 action="/action.php" specifies where form data will be sent.
 method="post" sends data securely.

12. <label>

Displays labels for form fields.

Example: <label for="username">Student Name</label>

Attribute:

 for connects the label to a specific input field.

13. <input>

Different input types used:

a) type="text"
Used for entering student name.

b) type="password"
Used for entering portal password.

Attribute:

 placeholder displays hint text.

c) type="checkbox"
Used for agreeing to terms and conditions.

d) type="radio"
Used for selecting gender.

Attribute:

 name="gender" groups radio buttons together.

e) type="file"
Used for uploading a resume.

f) type="number"
Used for entering age.

Attributes:

 min="18"
 max="100"

g) type="range"
Creates a slider for interest level.

h) type="date"
Used for selecting date of birth.

i) type="email"
Used for entering an email address.

j) type="url"
Used for entering a LinkedIn or profile URL.

k) type="search"
Used for searching a course.

Common Input Attributes:

 id : Unique identifier.
 name : Name used when submitting data.
 placeholder : Displays hint text.
 value : Value submitted with the form.

14. <select>

Creates a dropdown menu for course selection.

Attribute:

 name="course"

15. <option>

Represents an item inside the dropdown list.

Options:

 Web Development
 Data Science
 Cyber Security
 Cloud Computing

16. <textarea>

Creates a multi-line text field.

Attributes:
 rows="4"
cols="30"

Used for:
"Why do you want to enroll?"

17. <button type="submit">

Creates a submit button.

Attribute:

 type="submit"

Button Text:
Enroll Now

CSS PROPERTIES USED

1. font-family
   Sets the font style of the webpage.

2. background
   Creates a gradient background using two colors.

3. display: flex
   Uses Flexbox layout.

4. justify-content: center
   Centers the form horizontally.

5. align-items: center
   Centers the form vertically.

6. min-height: 100vh
   Makes the page cover the full screen height.

7. width
   Sets the width of the form container.

8. padding
   Adds spacing inside the container.

9. border-radius
   Creates rounded corners.

10. box-shadow
    Adds shadow around the form.

11. text-align
    Aligns text to the center.

12. color
    Changes text color.

13. display: block
    Displays labels on separate lines.

14. margin-top
    Adds space above elements.

15. margin-bottom
    Adds space below elements.

16. border
    Creates borders around form fields.

17. cursor: pointer
    Changes the mouse cursor when hovering over the button.

18. font-size
    Sets the button text size.

19. button:hover
    Changes button color when the mouse pointer is placed over it.


PROJECT SUMMARY

This project is a Student Course Enrollment Form developed using HTML and CSS. The form allows students to enter personal details, select a course, upload a resume, and submit enrollment information. CSS has been used to create a clean and user-friendly interface.
